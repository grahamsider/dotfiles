## Info

- Window Manager: `i3-gaps`
- Terminal: `st`
- Compositor: `compton`
- Text Editor: `neovim`
- Bar: `polybar`
- dmenu: `rofi`

Other notable packages: `pywal`, `neofetch`
(Check out [dylanaraps profile](https://github.com/dylanaraps). His setups have been a big inspiration for this!)
