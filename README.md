## Info:

- Window Manager: `i3-gaps`
- Terminal: `compton`
- Text Editor: `neovim`
- Bar: `polybar`
- dmenu: `rofi`

Other notable packages: `pywal`, `neofetch`
(Check out [dylanaraps profile](https://github.com/dylanaraps). His setup was a big inspiration for this :simple_smile:)
